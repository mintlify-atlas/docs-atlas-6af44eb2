The Authentication service provides production-ready user authentication using JWT tokens, bcrypt password hashing, and FastAPI's OAuth2 security primitives.

## Overview

The authentication service includes:

- **JWT token generation and validation** with configurable expiration
- **Secure password hashing** using bcrypt with automatic truncation
- **User management** with create, read, update, and delete operations
- **Account activation/deactivation** for user lifecycle management
- **OAuth2-compatible endpoints** for standard authentication flows

<Note>
The auth service requires the **backend** and **database** components. These dependencies are automatically included when you add the service.
</Note>

## Adding Authentication to Your Project

Add the authentication service to a new or existing project:

<CodeGroup>
```bash New Project
aegis init my-app --services auth
cd my-app
```

```bash Existing Project
aegis add-service auth --project-path ./my-app
```
</CodeGroup>

The service includes:

- User models (`app/models/user.py`)
- Authentication service logic (`app/services/auth/`)
- Security utilities (`app/core/security.py`)
- API routes (`app/components/backend/api/auth/router.py`)
- Database migrations for the User table

## Configuration

Configure authentication settings in your `.env` file:

```bash .env
# Secret key for JWT signing (generate with: openssl rand -hex 32)
SECRET_KEY=your-super-secret-key-change-this-in-production

# JWT algorithm
JWT_ALGORITHM=HS256

# Token expiration in minutes
ACCESS_TOKEN_EXPIRE_MINUTES=43200  # 30 days
```

<Warning>
**Never commit your SECRET_KEY to version control.** Generate a secure key for each environment:

```bash
openssl rand -hex 32
```
</Warning>

## Usage Examples

### Protected Routes

Use the authentication service to protect your API endpoints:

```python app/components/backend/api/your_router.py
from fastapi import APIRouter, Depends
from app.services.auth.auth_service import get_current_user_from_token
from app.models.user import User
from fastapi.security import OAuth2PasswordBearer

router = APIRouter()
oauth2_scheme = OAuth2PasswordBearer(tokenUrl="/api/v1/auth/token")

@router.get("/protected")
async def protected_route(
    token: str = Depends(oauth2_scheme),
    db: AsyncSession = Depends(get_async_db)
):
    user = await get_current_user_from_token(token, db)
    return {"message": f"Hello {user.email}"}
```

## Security Features

<Steps>
  <Step title="Password Hashing">
    Passwords are hashed using bcrypt with automatic salt generation. The service includes automatic truncation to 72 bytes for bcrypt compatibility.
    
    ```python app/core/security.py
    def get_password_hash(password: str) -> str:
        """Hash a password using bcrypt."""
        truncated = password[:72]  # bcrypt limit
        return bcrypt.hashpw(
            truncated.encode("utf-8"), 
            bcrypt.gensalt()
        ).decode("utf-8")
    ```
  </Step>
  
  <Step title="JWT Token Validation">
    Tokens are validated on every protected request, checking signature and expiration.
    
    ```python
    def verify_token(token: str) -> dict[str, Any] | None:
        """Verify and decode a JWT token."""
        try:
            payload = jwt.decode(
                token, 
                settings.SECRET_KEY, 
                algorithms=[settings.JWT_ALGORITHM]
            )
            return payload
        except JWTError:
            return None
    ```
  </Step>
  
  <Step title="Account Status Checks">
    Inactive users are automatically rejected during authentication, even with valid tokens.
    
    ```python
    if not user.is_active:
        raise HTTPException(
            status_code=401,
            detail="Inactive user"
        )
    ```
  </Step>
</Steps>

## Integration Patterns

### Role-Based Access Control (RBAC)

Extend the User model to support roles:

```python
# Add to app/models/user.py
from enum import Enum

class UserRole(str, Enum):
    ADMIN = "admin"
    USER = "user"
    GUEST = "guest"

class User(UserBase, table=True):
    # ... existing fields ...
    role: UserRole = Field(default=UserRole.USER)

# Create role dependency
def require_role(required_role: UserRole):
    async def role_checker(
        token: str = Depends(oauth2_scheme),
        db: AsyncSession = Depends(get_async_db)
    ):
        user = await get_current_user_from_token(token, db)
        if user.role != required_role:
            raise HTTPException(403, "Insufficient permissions")
        return user
    return role_checker

# Use in routes
@router.delete("/admin/users/{user_id}")
async def delete_user(
    user_id: int,
    admin: User = Depends(require_role(UserRole.ADMIN))
):
    # Only admins can delete users
    pass
```

### Multi-Factor Authentication (MFA)

Add TOTP-based two-factor authentication:

```python
import pyotp

# Generate MFA secret during registration
def generate_mfa_secret() -> str:
    return pyotp.random_base32()

# Verify MFA code during login
def verify_mfa_code(secret: str, code: str) -> bool:
    totp = pyotp.TOTP(secret)
    return totp.verify(code, valid_window=1)
```

## Testing

Test authentication flows in your application:

```python tests/test_auth.py
import pytest
from httpx import AsyncClient

@pytest.mark.asyncio
async def test_user_registration(client: AsyncClient):
    """Test user registration endpoint."""
    response = await client.post(
        "/api/v1/auth/register",
        json={
            "email": "test@example.com",
            "password": "testpass123",
            "full_name": "Test User"
        }
    )
    assert response.status_code == 200
    data = response.json()
    assert data["email"] == "test@example.com"
    assert "id" in data

@pytest.mark.asyncio
async def test_login_flow(client: AsyncClient):
    """Test complete login flow."""
    # Register user
    await client.post(
        "/api/v1/auth/register",
        json={
            "email": "login@example.com",
            "password": "loginpass123"
        }
    )
    
    # Login
    response = await client.post(
        "/api/v1/auth/token",
        data={
            "username": "login@example.com",
            "password": "loginpass123"
        }
    )
    assert response.status_code == 200
    token_data = response.json()
    assert "access_token" in token_data
    
    # Access protected route
    token = token_data["access_token"]
    response = await client.get(
        "/api/v1/auth/me",
        headers={"Authorization": f"Bearer {token}"}
    )
    assert response.status_code == 200
```
